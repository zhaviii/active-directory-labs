## 📌 Purpose

Active Directory labs and documentation.

## Topics

- Domain setup and promotion
- Organizational Units (OUs)
- User and group management
- Group Policy Objects (GPOs)
- DNS integration with AD

## Key Concepts

| Term | Meaning |
|------|--------|
| DC | Domain Controller — the server managing AD |
| OU | Organizational Unit — folder for objects |
| GPO | Group Policy Object — settings applied to users/computers |
| LDAP | Protocol AD uses for directory queries |
